## Style Reference: EZ Beats & Bites (Rustic Garden / Boho)

**Tokens:**
- Colors:
  - Background (Light): `#FDFBF7` (Warm Cream / Off-White)
  - Primary (Brand): `#3B4A3F` (Deep Olive / Forest Green)
  - Secondary/Accent: `#E5B25D` (Warm Muted Gold for string lights/buttons)
  - Text (Primary): `#2C332D` (Dark Greenish Gray)
  - Text (Muted): `#637066` (Sage Gray)
- Type scale:
  - Headings (Serif): `Playfair Display`, serif (Matches the "EZ" elegant logo)
  - Accent/Script (Cursive): `Dancing Script`, cursive (Matches the "Beats and Bites" cursive)
  - Body (Sans): `Outfit` or `Inter`, sans-serif
- Spacing scale: Tailwind default
- Radii: `rounded-3xl` for soft organic shapes, `rounded-full` for buttons.
- Shadows / Effects: 
  - Soft, earthy drop shadows `shadow-[0_8px_30px_rgb(59,74,63,0.08)]`.
  - Minimal borders `border-[#E2DFD2]`.

**Components styled:**
| Component | Default | Hover | Active |
|---|---|---|---|
| Primary Button | bg-[#3B4A3F] text-[#FDFBF7] rounded-full | bg-[#2C362F] scale-105 | scale-95 |
| Accent Button | bg-[#E5B25D] text-[#2C332D] rounded-full | bg-[#D4A352] | scale-95 |
| Menu Card | bg-white rounded-3xl border border-[#E2DFD2] text-[#2C332D] | shadow-xl translate-y-[-4px] | scale-[0.98] |

**Contrast checked:** Yes, Dark Green on Cream is excellent for reading.
